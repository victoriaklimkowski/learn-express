# Express Tutorial

## Setup
- Run `npm install` from the project root.
- Run `npm install` from `ui-app/`.

To run any server use `node /path/to/server` from the project root.

Updated:
`npm install -D ts-node`
`npx ts-node {path_to_server}/server.ts`
